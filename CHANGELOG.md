# Change Log

All notable changes to the "vscode-gemini-cli" extension will be documented in this file.

## [0.1.0] - 2024-03-06

### Added
- Initial release of Gemini CLI extension
- WebView-based chat interface similar to Claude Code
- Real-time streaming responses from Gemini CLI
- Syntax highlighting for code blocks
- Code actions: copy and insert code
- Multiple model support (Gemini Pro, Pro Vision, Ultra)
- Configurable temperature and max tokens
- Keyboard shortcuts (Ctrl+Shift+G / Cmd+Shift+G)
- Sidebar integration with quick actions
- Auto-scroll to latest message
- CLI availability checking
- Welcome screen with suggested prompts

### Features
- 🤖 AI-powered chat interface
- 💻 Code generation and explanation
- 🎨 Beautiful syntax highlighting
- ⚡ Real-time response streaming
- 📋 Copy/insert code functionality
- 🎯 Multiple Gemini models
- ⌨️ Keyboard shortcuts
- 🌙 VSCode theme integration
